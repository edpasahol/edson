My Reflection on CSS Dimensions and Pseudo-Classes

(Lesson 6: Setting Dimensions and Link States)

What I Have Learned

In this lesson, I learned that CSS can be used to manage the dimensions of web elements such as width, height, and visibility. These properties allow us to control the layout and appearance of different parts of a webpage, making them more flexible and organized.

The important properties I learned include:

  • visibility – determines whether an element is visible or hidden.
  • width and height – control the size of an element.
  • max-width / min-width – define the maximum and minimum width.
  • max-height / min-height – define the maximum and minimum height.
  • line-height – adjusts the spacing between lines of text.

I also learned about CSS Pseudo-Classes, which are special selectors that define the behavior of elements under certain conditions. These are commonly used with links to control how they look when interacted with.

The four main link pseudo-classes are:

  • :link – for unvisited links.
  • :visited – for links that have been clicked before.
  • :hover – when the mouse pointer is over a link.
  • :active – when the link is being clicked.

I discovered that the correct order to write these is LVHA (Link, Visited, Hover, Active) to make sure all link effects display properly. I also learned that pseudo-classes are not only for links — they can be used for images, paragraphs, and other elements to make websites more interactive and dynamic.

Overall, this lesson helped me understand how to control both the structure and behavior of webpage elements using CSS. By mastering dimensions and pseudo-classes, I can make web pages that are both well-designed and user-friendly.
