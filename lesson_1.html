My Reflection on CSS Introduction

(Lesson 1: Cascading Style Sheets)

What I Have Learned

In this lesson, I learned that CSS stands for Cascading Style Sheets. It is used to control the presentation and design of a webpage, such as the layout, colors, fonts, and spacing. CSS allows web developers to separate the content (HTML) from the design, making web pages easier to maintain and more visually appealing.

I also learned that there are three layers of a webpage: the Content Layer (HTML), the Presentation Layer (CSS), and the Behavior Layer (JavaScript). Each layer has its own role in creating a complete and functional website.

One of the most important benefits of using CSS is that it saves time and ensures consistency across multiple webpages. A single CSS file can style many pages at once, which helps maintain a uniform look and makes it easier to update the design later.

Overall, this lesson helped me understand that CSS is essential in creating modern, responsive, and attractive websites. It made me appreciate how web developers combine structure and style to create functional and beautiful pages.
