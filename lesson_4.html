My Reflection on CSS Rules and Classes

(Lesson 4: CSS RuleS, Properties, and Class Selectors)

What I Have Learned

In this lesson, I learned that a CSS Rule consists of two main parts: the selector and one or more style declarations. The selector identifies which HTML element to style, while the declaration specifies what visual changes will be applied using properties and values.

I also learned about the six main groups of style properties, which include:

  • Fonts
  • Colors and Backgrounds
  • Text
  • Boxes and Layout
  • Lists
  • Tag Classifications

The lesson also introduced the concept of CSS Classes, which are reusable style definitions used to format specific elements. Classes are defined using a dot (.) followed by the class name — for example, .highlight. Classes allow us to style multiple elements in a consistent way.

I learned that there are two ways to use class selectors:

  • .classname — applies to any element with that class.
  • element.classname — applies only to that specific element type with the class.

Lastly, I discovered that we can group selectors using commas to apply the same style to multiple elements, which makes the code cleaner and easier to manage. For example: h1, p { color: blue; } styles both headers and paragraphs at once.

Overall, this lesson taught me how to create flexible and organized CSS code. I now understand how to use classes to make webpages more dynamic, consistent, and professional-looking.
