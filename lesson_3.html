My Reflection on Display Properties

(Lesson 3: Display Properties and Div & Span Tags)

What I Have Learned

In this lesson, I learned about the different ways to style web pages using CSS and how HTML tags like <span></span> and <div></div> help structure and format content.

I understood that the <div> tag is used for larger sections of content and creates block-level elements that break lines, important for layout sections.

The <span> tag is used for small chunks of inline content, which can be styled differently without breaking the flow of text.

I also learned about block and inline elements. Block elements like the <div> tag stack vertically, while inline elements like the <span>> tag flow horizontally within sentences. This understanding helps me decide when to use each tag for page layouts.

Using CSS, I learned I can control how these elements appear and behave, for example, using the display property to change inline to block or vice versa.
