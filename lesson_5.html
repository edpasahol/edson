My Reflection on CSS Selectors

(Lesson 5: CSS Selectors)

What I Have Learned

Element/Tag Selector: Styles all elements of a specific type, e.g., h1 or p.
Class Selector: Applies styles to multiple elements sharing the same class attribute. Use .classname.
ID Selector: Styles a unique element with a specific id using #idname.
Universal Selector: Targets all elements on the page with *.
Group Selector: Applies styles to multiple elements by separating selectors with commas, e.g., h1, h2, p.
Attribute Selector: Styles elements based on attributes, e.g., p[class].

Learning about CSS selectors has made me realize how powerful and flexible CSS is. Different selectors allow me to target HTML elements in precise ways—whether it's styling all paragraphs, specific sections, or elements with certain attributes. I now understand how combining selectors—like classes and IDs—helps create organized, maintainable stylesheets. This knowledge will help me design more efficient and visually appealing web pages in the future.
