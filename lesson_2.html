My Reflection on CSS Style Sheets

(Lesson 2: Inline, Embedded, and External CSS)

What I Have Learned

In this lesson, I learned that a Style Sheet is a set of instructions that tells the browser how to display elements on a webpage. CSS uses selectors to target HTML elements, and each selector can have multiple declarations containing properties and values that define how an element should look.

I also learned that there are three main types of CSS:

  • Inline Style Sheets – used directly inside an HTML tag using the style attribute. They affect only that specific element.
  • Embedded (Internal) Style Sheets – written inside the <style> tag within the <head> section. They apply styles to the whole document.
  • External Style Sheets – written in a separate .css file and linked using the <link> tag. They allow multiple pages to share the same design.

I discovered that the use of external CSS is the most efficient method since it separates the content from the design, making it easier to maintain and update multiple webpages at once. Embedded CSS is helpful for styling a single page differently, while inline CSS is best for small, specific changes.

Lastly, I learned that comments can be added in CSS using /* ... */ to explain code and make it easier to read. This helps in maintaining clear and organized stylesheets.

Overall, Lesson 2 helped me understand how CSS rules are written and how different kinds of style sheets affect a webpage. It made me appreciate how structure and design work together to make websites clean, consistent, and easy to modify.
